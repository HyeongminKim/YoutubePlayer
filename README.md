### 사용된 레파지토리
##### 여기 나열된 레파지토리를 만드신 분들이 없었다면 이 앱을 제작할 시도조차 못 했을 겁니다. 
- [youtubedl-android](https://github.com/yausername/youtubedl-android)
- [youtube-dl](https://github.com/ytdl-org/youtube-dl)
- [FFmpeg](https://github.com/FFmpeg/FFmpeg)
